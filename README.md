# @paybase/machine

An asynchronous finite state machine library.

### Installation

This library requires `async/await` and `Proxy` support in your environment, so ideally `node>=7.4`.

```
$ npm install --save @paybase/machine
```

or

```
$ yarn add @paybase/machine
```
